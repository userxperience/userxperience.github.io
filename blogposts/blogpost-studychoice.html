my study choice

The reason I like this study so far is because I enjoy the freedom it gives you. I graduated from middle school only last year so this is a big contrast to my previous experiences. I try to only ask the teachers if it is absolutely necessary because I like solving issues on my own. I'm also interested in learning more about how things in the ICT field work and I've really enjoyed creating this website as well.
Another thing I like is that this course has given me real information so far, this rather amateurish looking website is created using the same language as websites like Youtube, which is why this course actually feels useful
I remember opening Inspect Element in middle school, changing some text, and impressing my friends with my hacking skills, but now I actually understand how most of HTML works and it feels good to have the skill to create my own site, amateurish as it may be.
